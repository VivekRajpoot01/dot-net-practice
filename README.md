## 🚀 Getting Started

### Prerequisites

- [.NET SDK 8.0+](https://dotnet.microsoft.com/download) installed
- [Visual Studio 2022](https://visualstudio.microsoft.com/) (recommended) or VS Code
- [SQL Server](https://www.microsoft.com/en-us/sql-server/sql-server-downloads) (for database projects)

### Clone the Repository

```bash
git clone https://github.com/VivekRajpoot01/dot-net-practice.git
cd dot-net-practice
```

### Run a Console Application

```bash
cd ConditionalPractice
dotnet run
```

### Run a Web Application

```bash
cd MVC_Core_WebApp/MVC_Core_WebApp
dotnet run
```

Then open your browser and navigate to `https://localhost:5001` or `http://localhost:5000`.
