
Scenario: George Finance Company
George Finance Company has an application for providing loans to their customers. They provide only Personal Loan and Home Loan. You need to help them generate the EMI (Equated Monthly Installment) based on the user's choice.
Requirements
1. Class Loan
• Fields (Private):
    ◦ loanNumber (string)
    ◦ loanProviderName (string)
    ◦ loanAmount (double)
• Properties (Public):
    ◦ LoanNumber (string)
    ◦ LoanProviderName (string)
    ◦ LoanAmount (double)
2. Class PersonalLoan (Inherits Loan)
• Fields (Private):
    ◦ numberOfYears (int)
    ◦ interestRate (float)
• Properties (Public):
    ◦ NumberOfYears (int)
    ◦ InterestRate (float)
• Method:
    ◦ public double CalculateEMI()
    ◦ Formula: 
EMI= 
NumberOfYears×12
LoanAmount+(LoanAmount×NumberOfYears×InterestRate×0.01)
​
 
3. Class HomeLoan (Inherits Loan)
• Fields (Private):
    ◦ houseAge (int)
    ◦ interestRate (float)
• Properties (Public):
    ◦ HouseAge (int)
    ◦ InterestRate (float)
• Method:
    ◦ public double CalculateEMI()
    ◦ Formula: 
EMI= 
HouseAge×12
LoanAmount+(LoanAmount×InterestRate×0.01)
​
 
4. Class Program
• Method Main(string[] args):
    1. Get inputs: Loan Number, Loan Provider Name, Loan Amount.
    2. Get input: 1.Personal Loan or 2.Home Loan.
    3. If 1: Get Number of Years and Interest Rate. Create a PersonalLoan object and set all properties.
    4. If 2: Get House Age and Interest Rate. Create a HomeLoan object and set all properties.
    5. Call the AddLoan method (see below), passing the object and the option (1 or 2).
• Method AddLoan(Loan loan, int opt):
    ◦ If opt is 1: Cast the loan object to PersonalLoan and call CalculateEMI.
    ◦ If opt is 2: Cast the loan object to HomeLoan and call CalculateEMI.
    ◦ Output: Print exactly "Calculated EMI : [Result]" (Sample output shows result as integer, e.g., 13333).
